| Class                  | Description                              |
| ---------------------- | ---------------------------------------- |
| `display-block`        | Sets `display: block;` on element        |
| `display-inline-block` | Sets `display: inline-block;` on element |
| `display-table`        | Sets `display: table;` on element        |
| `display-table-cell`   | Sets `display: table-cell;` on element   |
